### ============== Installing Artillery 2 ===============

1) Install NodeJS server:
```
sudo apt-get install nodejs
```
2) Install Artillery 2:
```
sudo npm install -g artillery
```
3) Check that the installation succeeded:
```
artillery dino
```
4) Run load test:
```
artillery run < some_load_test.yml>
```
